- **Keep**
    
    이번주차에서 어떤 일이 특별했던 이유는? 
    -> 처음 필드미팅, 첫 오프라인 트랙세션. 처음 만났기 때문에 특별했다. 
    
    계속 하고 싶거나, 더 하고 싶은 것은?
    -> css 더 알고 싶다!
    
    성공의 요인이 되었던 것은?
    -> 모르는 것을 계속 물어봤다. 
    
- **Problem**
    
    반복해서는 안될 일은 무엇일까?
    -> 좌절하지 않기
    
    후회하는 일은? 그 원인은 무엇일까?
    -> 다른 사람과 비교하며 좌절했다. 어렵기도 하고, 잘 못하는 것 같아 걱정됐다. 
    
    실패와 힘듦의 요인이 되었던 것은?
    -> 자신감 하락. 
    
- **Try**
    
    앞선 단계에서 제기된 problem들 중 하나에 대한 해결책이 있다면 모두 적는다.
    -> 모르는 것이 있으면 바로바로 물어보기 
    -> 자신감을 잃지 말기
    -> 계속 시도해보기
